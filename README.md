# Copilot Studio SSO GitHub Pages

This project provides documentation for configuring Single Sign-On (SSO) with Copilot Studio. It serves as a guide for users to set up and troubleshoot SSO integration effectively.

## Project Structure

- **.github/workflows/pages.yml**: GitHub Actions workflow for deploying the GitHub Pages site.
- **docs/index.md**: Main entry point for the documentation.
- **docs/sso/index.md**: Documentation specific to configuring SSO.
- **docs/sso/troubleshooting.md**: Troubleshooting tips for SSO configuration issues.
- **docs/assets/styles.css**: CSS styles for the documentation site.
- **_config.yml**: Configuration file for the GitHub Pages site.
- **Gemfile**: Specifies Ruby gems required for the project.
- **.gitignore**: Lists files and directories to be ignored by Git.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the required dependencies by running `bundle install`.
4. Serve the documentation locally using `bundle exec jekyll serve`.
5. Access the documentation at `http://localhost:4000`.

## Usage

Refer to the documentation in the `docs` directory for detailed instructions on configuring SSO and troubleshooting common issues.
